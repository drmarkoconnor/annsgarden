# Future Features

These features should not be built in MVP unless explicitly moved into scope.

## Version 2 candidates

### RHS and external guidance

- curated RHS links
- external guidance links by plant/problem
- treatment references
- no long copied content from external sites

### Weather

- live weather integration
- frost warnings
- rain-aware watering prompts
- wind warnings for spraying
- weather-based task suitability

### Calendar

- Apple Calendar integration
- export selected tasks to calendar
- optional personal reminders

### Offline support

- stronger offline editing
- sync when connection returns
- offline photo queue
- offline task completion queue

### Visual garden map

- simple map of garden areas
- tappable zones
- area photo markers

### AI features

- plant identification from photos
- pest/disease suggestion from photos
- photo comparison
- seasonal gap detection
- “what changed since last month?” summaries

### Native app

- investigate native iPhone app later
- only consider if the web app proves insufficient

### Public product

- support multiple gardens
- onboarding for other gardeners
- configurable locations
- generic task templates
- paid product possibilities

### Hard landscaping

- detailed records for paving, fences, gates and repairs
- recurring repair/inspection tasks

### Furniture

- furniture records
- cleaning/oiling/storage reminders

Currently excluded from MVP.

### Shopping list

- dedicated shopping list
- quantities
- suppliers
- purchase status

For MVP, purchase needs stay as notes inside planned tasks.

### Greenhouse

- greenhouse tasks
- ventilation
- cleaning
- seed sowing
- overwintering

Only add when Ann has a greenhouse.

### Reporting

- monthly summary
- seasonal summary
- annual review
- lessons for next year
- repeated issues report
- plant performance review
